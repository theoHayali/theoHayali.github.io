Homework 7 - Objects

Part 1

Web page with an input box for the user to enter a number with 4 decimals , a button to submit the data a button to clear the form.

Each of these input types will implement error checking validation. Then will calculate and display the following:
  1. Round the floating-point number to the nearest integer.
  2. Calculate the square root of the floating-point number and round it to an integer.
  3. Round the floating-point number to the nearest tenths position.
  4. Round the floating-point number to the nearest hundredths position.
  5. Round the floating-point number to the nearest thousandths position.

Part 2

Web page with a form and a textarea to allow the user to type a long content an input box to allow the user to type only one letter:

The user will type whatever content he/she wants in the textarea for the long content, and the user will also type a single character in the input box you provided. When the user clicks the submit button, you will: Search for the character the user typed inside the content in the textarea and you will count how many times this character is shown in the big content If the character is found, you will present a message in the output/result area that says how many times the character X shows up in the content where X is the single character typed by the user. If the character is not found, you will present a new window (using the window.open() method you learned), the message: "Search character X not found in the content you typed", where X is the single character typed by the user

Part 3

Web page with a form and input text box -The user needs to type the telephone number, as a string, in this format (999) 999-9999. You should use regular expression to validate the phone number. If the phone number typed is not in the correct format, you will present an error message in the textarea or paragraph you created showing to the user the correct way to type the phone number. If the phone number typed is in the correct format, you will present a message in the textarea or paragraph you created thanking the user for his/her phone number and that the user should expect to be contacted by one of the agents in the future.
