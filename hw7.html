Part 2

Web page with a script that will save TWO pairs of cookies for the same page - one of the pairs for the name of the user and another one for the username of the user

The Web page will work as the folllowing:


If it is the first time the user is opening the page on the browser, you will present a form for the user to type his/her name and his/her username. If it is not the first time the user is opening the page (if he/she is revisiting the page) while the browser is still opened, you will just display a message of Welcome Back followed by the name of the user that you will retrieve from the cookie previously saved.
